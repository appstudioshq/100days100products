# snake game

# description

- snake game is a game where the player controls a snake that moves around a grid. The snake eats food that appears randomly on the grid. When the snake eats food, it grows longer. The game ends when the snake collides with itself or with the walls of the grid. The player's score is the length of the snake.

# technology
- html
- css
- javascript

# features
- snake moves in 4 directions
- snake grows when it eats food
- game ends when snake collides with itself or walls
- score is displayed
- game can be paused and resumed
- game can be restarted
- game can be played in full screen

# ui 
- classic UI (Nokia 3310 style)
